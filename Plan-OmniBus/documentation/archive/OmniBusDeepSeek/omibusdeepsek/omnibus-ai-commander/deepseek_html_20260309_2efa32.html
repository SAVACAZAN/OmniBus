🚀 OmniBus Visual Installer v1.0

Acesta este asistentul tău personal pentru instalarea sistemului de trading de mare frecvență OmniBus. Urmează pașii în ordine și totul va funcționa perfect! 🎯

Pasul 1 din 8: Verificare sistem
Început 0% Final
1
🔍 Verificare sistem
Verificăm dacă sistemul tău suportă virtualizare și are toate uneltele necesare.
egrep -c '(vmx|svm)' /proc/cpuinfo
🔍 Interpretare rezultat:
• Dacă vezi un număr MAI MARE CA 0 (ex: 4, 8) → poți continua
• Dacă vezi 0 → trebuie să activezi virtualizarea în BIOS (Intel VT-x sau AMD-V)

📝 Cum activezi în BIOS:
1. Restartează calculatorul
2. Apasă F2, Del sau F10 la pornire
3. Caută "Intel VT-x", "AMD-V" sau "Virtualization Technology"
4. Activează și salvează (F10)
2
📦 Instalare KVM și unelte
Instalăm hypervisor-ul KVM și uneltele de virtualizare.
sudo apt update && sudo apt install -y qemu-kvm libvirt-daemon-system libvirt-clients bridge-utils virt-manager
sudo usermod -aG libvirt $USER && sudo usermod -aG kvm $USER
⚠️ IMPORTANT: După ce rulezi comenzile, trebuie să te DECONECTEZI și să te RECONECTEZI (să restartăm calculatorul) pentru ca modificările să aibă efect!
3
🔧 Instalare Zig și NASM
Instalăm compilatoarele necesare pentru OmniBus.
sudo apt install -y nasm make wget
wget https://ziglang.org/download/0.13.0/zig-linux-x86_64-0.13.0.tar.xz && tar -xf zig-linux-x86_64-0.13.0.tar.xz && sudo mv zig-linux-x86_64-0.13.0 /opt/zig && sudo ln -s /opt/zig/zig /usr/local/bin/zig
4
📥 Clonare OmniBus
Descărcăm codul sursă al proiectului.
cd ~ && git clone https://github.com/SAVACAZAN/OmniBus && cd OmniBus
5
🛠️ Compilare OmniBus
Compilăm bootloader-ul și modulele.
make clean && make build
ls -la *.img
✅ Ar trebui să vezi un fișier .img (ex: omnibus.img)
6
🎮 Test în QEMU
Testăm rapid dacă imaginea funcționează în emulator.
make qemu
📝 Să vezi o fereastră neagră cu text. Ca să ieși: Ctrl + A, apoi X
7
💻 Creare VM în KVM
Creăm mașina virtuală în Virtual Machine Manager.
virt-manager
📝 Pași în virt-manager:
1. Click "Create new VM"
2. Alege "Import existing disk image"
3. Browse → navighează la omnibus.img
4. RAM: 2048 MB, CPU: 2
5. Nume: "OmniBus"
6. Bifează "Customize configuration"
7. La Boot Options, bifează "Enable boot menu"
8. Click "Begin Installation"
8
🎉 Verificare finală
Verificăm dacă totul funcționează.
sudo systemctl status libvirtd
🎉 FELICITĂRI! Ai instalat OmniBus cu succes!
