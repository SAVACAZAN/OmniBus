================================================================================
MEMORY ADDRESSES READ BY DASHBOARD
================================================================================

0x110000 (256B) - Grid OS State
   - Grid metrics: magic, pair_id, flags, bounds, step, profit, tsc
   - Panel: TRADING
   - Update: Every 100ms

0x150000 (256B) - Analytics OS State
   - Price consensus data
   - Panel: TRADING (used for arb detection)
   - Update: Every 100ms

0x310000 (128B) - Checksum OS State
   - Integrity verification metadata
   - Panel: HEALTH
   - Update: Every 100ms

0x320000 (128B) - AutoRepair OS State
   - Repair tracking, failure count
   - Panel: HEALTH
   - Update: Every 100ms

0x330000 (128B) - Zorin OS State
   - Access control, zones, violations
   - Panel: COMPLIANCE
   - Update: Every 100ms

0x340000 (512B) - Audit Log OS
   - Recent 16 events with timestamps
   - Panel: AUDIT
   - Update: Every 100ms

0x2D0000 (128B) - NeuroOS State
   - Generation, evolution cycles, fitness scores
   - Panel: NEURO
   - Update: Every 100ms

0x400000 (512B) - OmniStruct (optional)
   - All Tier 1 aggregates
   - Not currently used (future)
